# 🤖 AI Automation Agent

## 🚀 Overview

This project is a simple AI-powered automation agent built using Python and Flask.
It can understand user input, fetch real-time weather data, and send automated email reports.

---

## ✨ Features

* 🌦️ Real-time weather data using OpenWeather API
* 📧 Email automation using SMTP
* 🧠 Intent detection (weather / email commands)
* 🌐 Simple web interface using Flask

---

## 🛠️ Tech Stack

* Python
* Flask
* OpenWeather API
* SMTP (Gmail)

---

## 📦 Installation

1. Clone the repository:

```
git clone https://github.com/yourusername/ai-automation-agent.git
cd ai-automation-agent
```

2. Install dependencies:

```
pip install flask requests
```

---

## 🔑 Configuration

Update the following in `main.py`:

```
WEATHER_API_KEY = "your_api_key"
sender = "your_email@gmail.com"
password = "your_app_password"
```

---

## ▶️ Run the Project

```
python main.py
```

Open in browser:

```
http://127.0.0.1:5000/
```
